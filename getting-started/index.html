Getting started#

If you are new to Ubermag this is the correct place to start learning it. The following set of tutorials will introduce all basic concepts of Ubermag and its subpackages.

  • Our first ubermag simulation
  • Magnetisation field
  • Energy equation
  • Dynamics equation
  • Exercise: Current induced domain wall motion
  • Exercise: Domain wall pair conversion

Where to go next?

There are several places where you can find more advanced documentation:

  • The Examples show examples of how to solve specific physical questions using ubermag.

  • The Package documentation contains explanations and examples of more technical functionality of the different packages, e.g. details about plotting or mathematical operations.

  • The API contains detailed documentation of all classes and functions of the different subpackages.
