Package documentation#

The notebooks shown here contain “technical” documentation for specific features of the different packages. If you are new to ubermag you should consider first looking at Getting started which will give a gentle introduction into all the packages inside ubermag.

Different calculators (OOMMF and mumax3) support different feature sets. The Compatibility: Ubermag - OOMMF - mumax3 page gives an overview over the different features. It also lists compatible versions for Ubermag, and OOMMF, and mumax3.
